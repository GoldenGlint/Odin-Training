Content first vs layout first design
- Content First Design: clarity how the content should be, and layout follows
- Use flexbox to control behaviour of items through logical rules.
- How they shrink grow, ideal size, position, dynamic layout
- General layout can change a lot

- Layout First Design: decide how you want the pieces arranged, fill the content
- grid shines as grid and column tracks give you full control of the layout
- content can only fill implicit and explicit

Combining flexbox and grid
- flexbox can make it easier to control how that content is positioned in a Flex container
- accurately place contet in complex way, Grid is easier
