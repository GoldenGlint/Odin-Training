Frameworks and Preprocessors:

Frameworks
    Bootstrap: A lot of heavy lifting packaging up common CSS code
    Tailwind: aim to change how we apply CSS through different css
    Ex: many framworks use .btn t oadd css
    Bulma and Foundation are becoming more popular
    Too many developers jump into frameworks too early in education
        Overriding a framework's styling or debugging on your page becomes difficult with no CSS fundamentals
        Can constrain projects

Preprocessors
    Languages that help you write CSS easily. Reduce code repitition and provide all sort of time saving code saving features
    Extensions to vanilla CSS that provide extra functionality
        Preprocessor takes your code, turns it into vanilla CSS to import into your project
    CSS nesting used to be preprocessors. 
    Common ones are SASS, LESS, Stylus. Used by 80% of people.
